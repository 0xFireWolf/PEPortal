FireWolf OS X PE 9

增强型 macOS 维护系统，现已推出全新版本

详细了解

再一次、回归初衷

FireWolf OS X PE 9，火狼 macOS PE 系列的第九个版本，是基于 macOS 原版镜像制作的增强型维护系统。PE 提供原生的 HFS 以及 APFS 文件系统的读写操作。内置的实用工具以及维护套件可帮助你在无法启动主系统时访问转移个人文件、恢复系统备份、修复系统磁盘权限、编辑 ACPI 表等。

了解新版功能

亮点功能

更强大

新版 FireWolf OS X PE 内置了多达 24 款应用程序与实用工具，提供全方面 macOS 系统维护的技术支持。

更灵活

为 PE 定制的多应用启动器现已支持应用管理。你无需手动修改启动镜像即可动态地导入导出应用程序。

更简洁

启动器的用户界面经过重新设计后，应用不再固定在主界面上。现在你可以更自由地排列以及为它们分类。

更方便

隆重介绍内置于新版 PE 的『火箭启动器』你现在可以直接拖入想打开的文件到“火箭”上就好了。快来试试！
