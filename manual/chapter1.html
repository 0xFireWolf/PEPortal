第一章 简述

第一章 简述

>> FireWolf OS X PE 是什么？

FireWolf OS X PE 也称 Mac PE，是基于 macOS 原版镜像制作的增强型维护系统。PE 提供原生的 APFS/HFS 文件系统的读写操作，其内置的实用工具和维护套件可帮助你在无法启动主系统时访问个人文件、恢复系统备份、编辑 ACPI 表以及修复权限重建缓存等。当前最新的 FireWolf OS X PE 9 为 PE 系列的第九个版本，同时支持白苹果和黑苹果。

>> FireWolf OS X PE 安全吗？

FireWolf OS X PE 与往常一样不包含任何广告、后门、垃圾推广等。你可以放心使用。

>> FireWolf OS X PE 在哪里下载？

我们强烈建议你从官方网站下载 PE，并验证镜像的 MD5 以及 SHA256 值。

请勿轻易从其他渠道下载，我们无法保证其安全性。

>> FireWolf OS X PE 有版本区分吗？我应该下载哪个版本？

FireWolf OS X PE 提供两大主要版本：包含 Clover 引导的版本 和 不包含 Clover 引导的版本。

包含引导的版本适合黑苹果新手使用，但我们无法保证内置的 Clover 启动配置文件 `config.plist` 兼容各种机器，必要的话请替换为适合你自己电脑的配置文件。

不包含引导的版本适合白苹果以及熟悉黑苹果的用户使用。该版本理论上仅支持 Clover 引导。如果你使用的变色龙的分支版本支持启动原版镜像的话，那么也应同样支持启动 PE 9。

>> FireWolf OS X PE 有必要吗？存在的意义是什么？

PE 这个名字其实是取自我们常用的 Windows PE，本意是方便安装和维护系统。

但是苹果自带的恢复分区（Recovery HD）实在是太简陋了，只有在线重装系统以及时光机恢复等几个功能。如果想从其他分区读写文件，只能通过终端敲命令来操作，这对于不熟悉命令行的一般用户不是特别友好。

和 Mac PE 的主要用户一样，我本人也是从黑果开始折腾的。那个时候兼容性不像现在这么好，每天改这改那的经常进不了系统。而当时 Windows 下那些比较流行的读写苹果分区软件（MacDrive，Paragon HFS+ 等）又不是那么靠谱，经常出现文件丢失、分区损坏等问题，所以 FireWolf OS X PE 的雏形版本只是移植了系统原生的 Finder 来方便转移文件。

现在苹果发布的 macOS High Sierra (10.13.x) 系统使用了新的 APFS 文件系统，而目前的 BootCamp 组件是不支持读写 APFS 分区的，所以 FireWolf OS X PE 的初衷再一次显现出来了。

比起完整的 macOS 系统，PE 仍然有很多局限性同时也是最大的短板：并不是所有软件都可以在 PE 下正常运行。

有些人可能觉得既然我都能正常启动 Mac PE 了，那我为什么不直接再安装个系统留着备用。对此我只能说可能每个人的看法和需求不一样吧。

现在黑苹果的稳定性和兼容性比以前提高了很多。数据无价，不怕一万就怕万一，PE 对我来说也就是留着备用，反而希望用到 PE 的那一天永远都不要到来。
