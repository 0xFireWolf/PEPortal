第四章 系统维护

第四章 系统维护

由于 PE 维护套件暂时停用，本章节将介绍如何通过命令行来修复磁盘权限以及重建内核缓存。

注意：请勿尝试导入 KCPM Utility Pro，该工具不支持在 PE 环境下运行。

修复磁盘权限

修复磁盘权限功能将由 RepairPermissions 命令行工具提供。关于该工具的介绍以及详细使用方法，请访问 RepairPermissions 的发布页面。

请通过启动器打开终端，输入 RepairPermissions --repair /Volumes/Macintosh\ HD回车即可。

请注意，--repair后跟目标系统分区的挂载点，不一定是 Macintosh HD。请根据实际情况来修改。

推荐直接从 Path Finder 里把目标分区拖到终端里，这样终端会帮你自动转义特殊字符。

重建内核缓存

重建内核缓存功能由系统自带的 kextcache 命令行工具提供。关于该工具的介绍，请自行查阅对应的手册文档（man page）。

请通过启动器打开终端，输入 kextcache -i /Volumes/Macintosh\ HD 回车即可。

请注意，-i后跟目标系统分区的挂载点，不一定是 Macintosh HD。请根据实际情况来修改。

推荐直接从 Path Finder 里把目标分区拖到终端里，这样终端会帮你自动转义特殊字符。
